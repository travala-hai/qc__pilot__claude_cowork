# TC-023 — Verify search "from" price matches lowest available package on detail page

- **ID:** TC-023
- **Title:** Verify search-card "from" price equals the lowest package price on the hotel detail page
- **URL(s) under test:** https://www.travala.com/search (Hanoi results) → hotel detail page
- **Preconditions:** Hanoi search results loaded (run TC-010 as setup); currency USD (KB §7). Target: **InterContinental Hanoi Westlake by IHG**.
- **Known issue:** BUG-002 — search page and detail page can pull prices from different suppliers (EPS vs RateHawk/EnTravel), so the card may show a higher price than the best detail-page deal. This test is expected to catch it.

## Steps

1. [ACTION] Locate the "InterContinental Hanoi Westlake by IHG" card (scroll/paginate; use name filter if available). If not found, fail as test-data availability issue. [ASSUMED — locate strategy required for autonomous execution.]
2. [ACTION] Read and record the card's "from" price → `CARD_PRICE` (`US$ N.NN`).
3. [ACTION] Confirm the card's **CHOOSE ROOM** button is visible, then click it.
4. [ACTION] If the detail page opens in a new tab, switch to it. [ASSUMED — common OTA behavior; ambient handling.]
5. [ACTION] Wait for the detail page to load: hotel name header visible and packages section rendered. Fallback: 10 s (KB §8).
6. [ACTION] Read the header "from US$ N.NN" price → `HEADER_PRICE`.
7. [ACTION] Scroll through the entire packages/rooms list, expanding any collapsed room types, and record every package's total price for the selected stay; take the minimum → `MIN_PACKAGE`. [ASSUMED expansion step — collapsed packages would hide the true minimum.]
8. [CHECKPOINT] `HEADER_PRICE = MIN_PACKAGE` (exact match to the cent). Fail → report both values, continue. [ASSUMED — implied by "from" semantics; needed to localize where a mismatch originates.]
9. [CHECKPOINT] `CARD_PRICE = MIN_PACKAGE` (exact match to the cent). Fail → stop and report CARD_PRICE, HEADER_PRICE, MIN_PACKAGE and the delta (lost-conversion risk), flagging as BUG-002-class supplier mismatch.
